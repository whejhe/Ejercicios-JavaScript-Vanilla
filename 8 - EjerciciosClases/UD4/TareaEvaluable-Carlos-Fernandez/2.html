Debes realizar un carrusel de imágenes. Para ello debes tener una página principal y un iframe de forma que en el iframe se vayan cargando con las imágenes del carrusel. La ventana principal debe tener dos botones: (5 ptos.)  Botón iniciar: inicia la muestra de las imágenes, de tal manera que, cada 2 segundos muestre una nueva imagen (2 ptos.)  Botón parar: para el carrusel de imágenes, dejando fija la imagen que en ese momento este mostrando. (0,5 ptos.) En el iframe cuando se pulse sobre la imagen debe pasar a la siguiente imagen. (1 pto.) El orden de las imágenes debe respetarse y no puede reiniciarse, solo pararse. De forma que si la aplicación se cierra en una imagen debe aparecer esa imagen la siguiente vez que abramos la aplicación. (1,5 ptos.) Debes hacer que el iframe al tamaño de la imagen. Recuerda que puedes usar el evento onClick de la etiqueta 'button'
