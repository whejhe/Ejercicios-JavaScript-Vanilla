El alcance (scope) en JavaScript se refiere a las reglas que determinan dónde y cómo las variables pueden ser accedidas dentro de tu código. El alcance define en qué partes del código una variable es visible y puede ser utilizada. JavaScript tiene dos tipos principales de alcance:

  • Alcance Global (Global Scope): Las variables declaradas fuera de cualquier función o bloque de código tienen un alcance global. Esto significa que estas variables son accesibles desde cualquier parte del código, incluyendo dentro de funciones y bloques.
  • Alcance Local (Local Scope): Las variables declaradas dentro de una función o bloque de código tienen un alcance local. Esto significa que solo son accesibles dentro de la función o el bloque en el que se declaran.
